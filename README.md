# AutoRoadAndFollowTarget
### 自动寻路和相机的跟随
###### 调整好相机和小球的位置，选择相机，然后GameObject->Align with view。首先将相机放置到小球下成为其子物体，
###### 由于小球放大了两倍，所以相机的便宜位置offset发大2倍，最后将相机从小球的下面移出。
